# GastroMetrics

## Descripción
Plataforma web SaaS para ayudar a emprendedores de comida a calcular costos, controlar inventario y analizar utilidades.

## Funcionalidades
- Gestión de recetas
- Cálculo de costos
- Control de inventario
- Registro de ventas
- Reportes

## Tecnologías
- Frontend: (React / Typescript)
- Backend: (Node.js)
- Base de datos: (MySQL)
